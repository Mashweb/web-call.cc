web-call.cc

Sequentially programmed web applications

The following are loadable programs:

  • simple_seq_webapp.scm
  • calculator.scm

To try one of them, type one of following commands into the BiwaScheme console, terminated by a RETURN:

  • (load "simple_seq_webapp.scm")
  • (load "calculator.scm")

The calculator program is slow to load but practically self explanatory. To understand how to use the "simple sequential web app" you'll have to read the code. Read the page "Sequentially Programmed Web App Demo" for more background on sequentially programmed applications.
