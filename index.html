The following are loadable programs:

  • simple_seq_webapp.scm
  • calculator.scm
To try one of them, type one of following commands into the BiwaScheme console, terminated by a RETURN:
  • (load "simple_seq_webapp.scm")
  • (load "calculator.scm")
The calculator program is slow to load but practically self explanatory. To understand how to use the "simple sequential web app" you'll have to read the code.

The following examples show how BiwaScheme can manipulate the DOM: 

(define elem #f)
(define children #f)
(set! elem (js-ref (js-invoke (getelem "body") "children") "0"))
elem ;; => [object HTMLDivElement]
(js-ref elem "children") ;; => [object HTMLCollection]
(set! elem (js-ref (js-invoke (getelem "body") "children") "0"))
